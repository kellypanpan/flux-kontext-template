## 📋 项目概览

**CreativeForge** 是一个基于Next.js 15的现代化创意内容平台，专为内容创作者、营销人员和企业打造。平台集成了AI驱动的内容生成、多格式导出、品牌一致性管理等功能。

### 🏗️ 技术栈
- **前端框架**: Next.js 15 + React 18 + TypeScript
- **UI组件**: Shadcn/ui + Radix UI + Tailwind CSS
- **数据库**: Supabase (PostgreSQL)
- **支付系统**: Stripe
- **安全验证**: Cloudflare Turnstile
- **AI服务**: Fal.ai (Flux模型)
- **部署平台**: Vercel

### 📊 项目统计
- **总代码文件**: 153个文件
- **主要语言**: TypeScript (95%), JavaScript (5%)
- **代码行数**: 约50,000+行
- **支持语言**: 12种语言 (中文、英文、日文、韩文等)
